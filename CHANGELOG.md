# Changelog

All notable changes to `visio/product` will be documented in this file.

## 1.0.0 - 2024-12-16

### Added
- Initial release
- Idea management system
- Comment system for ideas
- Importance levels (Not important, Nice-to-have, Important, Critical)
- User reactions to ideas
- FilamentPHP 3.x resource integration
- Soft deletes for ideas and comments
- Configurable user model and table names
- Search and filter capabilities
- Clean and simple UI
